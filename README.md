# Vibe3D

Простой 3D-редактор полигональных мешей, вдохновлённый MODO и LightWave3D. Написан на D с использованием OpenGL 3.3 Core Profile, SDL2 и Dear ImGui.

![vibe3d](https://github.com/user-attachments/assets/33f67c74-7ed0-40fa-92e6-379573164d1f)

## Возможности

- Полигональное редактирование в трёх режимах: **Vertices**, **Edges**, **Polygons** (переключение `1`/`2`/`3` или `Space`)
- Инструменты трансформации с интерактивным гизмо: **Move** (W), **Rotate** (E), **Scale** (R)
- **Box tool** (B) — интерактивное создание кубоидов: рисовка основания + экструзия высоты
- Численный ввод параметров трансформации в ImGui-панели свойств
- Прямоугольное выделение drag-рамкой; Shift — добавить, Ctrl — убрать
- Connected selection — заливка смежных компонентов (`]`)
- Subdivision Surface — алгоритм Catmull-Clark (Shift+D)
- Frame-to-fit: вся сцена (A) или только выделение (Shift+A)
- Адаптивный размер гизмо (экранное масштабирование); 9 ступеней размера (`-` / `=`)
- Геометрическое определение видимости: задние грани, рёбра и вершины не выделяются
- Snap-вращение с шагом 15° при Ctrl
- Отрисовка через OpenGL VBO/VAO с оптимизациями (GPU offset drag, частичный upload вершин)
- Запись событий в JSON (F1/F2) и воспроизведение (`--playback <file>`)
- HTTP-сервер для автоматизированного тестирования (`--test`)

## Управление

### Камера

| Действие | Клавиши / мышь |
|---|---|
| Орбита | Alt + ЛКМ |
| Панорама | Alt + Shift + ЛКМ |
| Зум | Ctrl + Alt + ЛКМ |
| Вписать всю сцену | `A` |
| Вписать выделение | Shift + `A` |

### Режимы и инструменты

| Действие | Клавиша |
|---|---|
| Режим Vertices | `1` |
| Режим Edges | `2` |
| Режим Polygons | `3` |
| Переключить режим | `Space` |
| Move tool (toggle) | `W` |
| Rotate tool (toggle) | `E` |
| Scale tool (toggle) | `R` |
| Box tool (toggle) | `B` |

### Выделение

| Действие | Клавиши / мышь |
|---|---|
| Выделить | ЛКМ / drag-рамка |
| Добавить к выделению | Shift + ЛКМ / drag |
| Убрать из выделения | Ctrl + ЛКМ / drag |
| Connected selection | `]` |

### Операции с мешем

| Действие | Клавиша |
|---|---|
| Subdivision Catmull-Clark | Shift + `D` |
| Уменьшить размер гизмо | `-` |
| Увеличить размер гизмо | `=` |
| Начать запись событий | `F1` |
| Остановить запись | `F2` |
| Выйти | `Esc` |

### Гизмо Move / Scale

| Действие | Мышь |
|---|---|
| Перемещение / масштаб по оси | Drag по стрелке (X / Y / Z) |
| Перемещение / масштаб в плоскости | Drag по кольцу (XY / YZ / XZ) |
| Свободное перемещение / равномерный масштаб | Drag по центру |
| Ограничить ось при движении в плоскости | Ctrl + drag по плоскости |

### Гизмо Rotate

| Действие | Мышь |
|---|---|
| Вращение вокруг оси | Drag по дуге (X / Y / Z) |
| Вращение вокруг оси камеры | Drag по внешней дуге |
| Snap 15° | Ctrl + drag |

## Сборка и запуск

Требуется [DUB](https://dub.pm/) и компилятор D (DMD или LDC), а также установленный SDL2.

```sh
dub build
./vibe3d
```

### Флаги запуска

| Флаг | Описание |
|---|---|
| `--test` | Запустить с HTTP-сервером для автотестирования |
| `--playback events.log` | Воспроизвести записанную сессию событий |
| `--no-http` | Запустить без HTTP-сервера |
| `--http-port 9090` | Указать порт HTTP-сервера (по умолчанию: 8080) |

## Тестирование

Тесты — это D-программы, которые через HTTP API управляют запущенным экземпляром vibe3d.

```sh
./run_test.d                    # все тесты
./run_test.d test_bevel         # один тест (или: bevel, tests/test_bevel.d)
./run_test.d bevel selection    # несколько
./run_test.d -v test_bevel      # подробный вывод
./run_test.d --keep             # оставить vibe3d запущенным после прогона
./run_test.d --no-build         # пропустить `dub build`
```

Файлы тестов: `tests/test_*.d`. Записанные сессии событий: `tests/events/*.log`.
